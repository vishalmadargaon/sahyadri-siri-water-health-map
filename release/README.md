# Sahyadri-Siri Demo APK

Download `sahyadri-siri-v1.0-demo.apk` and install it on an Android device.

This public demo APK does not include a private Gemini API key. Firebase Auth, profile, reports, map, dashboard, and chatbot fallback guidance are available. Build locally with `GEMINI_API_KEY` in `local.properties` for Gemini-powered image analysis and assistant responses.
